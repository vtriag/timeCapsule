A MODA COMO VOCÊ NUNCA VIU

The Fashion Club é uma cápsula do tempo digital criada no contexto do projeto TimeCapsule 3025. A proposta é registrar a moda do século XXI sob uma perspectiva estética e cultural, como se estivéssemos apresentando nossas escolhas de estilo para visitantes do futuro.

Este site foi desenvolvido como trabalho acadêmico com o uso exclusivo de HTML e CSS, explorando conceitos de layout responsivo, design editorial e storytelling visual. Cada detalhe foi pensado para refletir a atmosfera elegante e marcante da moda contemporânea.
